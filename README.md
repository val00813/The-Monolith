# The Monolith
This project is an interactive 3D audio-visual art installation built with Three.js and the Web Audio API, fully supported as a Progressive Web App (PWA).

### Description
The Monolith explores the tension between systemic oppression and individual resistance. Users are placed inside a claustrophobic digital space where four massive wireframe walls slowly and constantly close in, driven by an invisible "fatigue" metric. 

To survive the compression, users must use their voice. Speaking or shouting generates a physical force that pushes the heavy walls back, while simultaneously causing the central glowing particle entity—representing the individual's will—to violently expand, glitch, and emit intense light. It is a digital metaphor for finding your voice and fighting back against the invisible pressures of space and time.

### Interaction Guide
1. Allow **Microphone Permissions** when prompted (essential for the experience).
2. Remain silent to observe the spatial compression and the feeling of confinement.
3. Speak, sing, or shout to push the walls away and trigger the generative bloom effects.
